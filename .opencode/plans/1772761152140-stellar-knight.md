3. **Add helper function to find executable:**
   ```typescript
   function findExecutable(command: string): string | null {
       const pathEnv = process.env.PATH || '';
       const directories = pathEnv.split(path.delimiter);
       
       for (const dir of directories) {
           const fullPath = path.join(dir, command);
           try {
               fs.accessSync(fullPath, fs.constants.X_OK);
               return fullPath;
           } catch {
               // File doesn't exist or lacks execute permissions - skip
               continue;
           }
       }
       return null;
   }
   ```

## Key Points
- Use `path.delimiter` for OS-agnostic PATH splitting (`:` on Unix, `;` on Windows)
- Use `fs.accessSync(path, fs.constants.X_OK)` to check execute permissions
- Skip directories that don't exist or files without execute permissions
- Search PATH directories in order, return first match
